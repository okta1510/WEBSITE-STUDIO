## 1. Requirements
- XAMPP/WAMP/LAMP with PHP 7.4+

## 2. Installation Steps

### Step 1: Extract Files
1. Extract STUDIO_FOTO_CEKREK.rar to htdocs folder
2. Path should be: C:\xampp\htdocs\STUDIO_FOTO_CEKREK\

### Step 2: Database Setup
1. Start XAMPP (Apache + MySQL)
2. Open phpMyAdmin (http://localhost/phpmyadmin)
3. Create new database: 'studio_foto_cekrek'
4. Import database file: database_clean_original_[timestamp].sql

## 4. Common Issues & Solutions

### ❌ Wrong Packages Showing
Problem: You see "Portrait Session", "Family Package", "Wedding Package" instead of Solo/Duo/Trio
Solution: 
- You imported the wrong database file
- Delete the database in phpMyAdmin
- Create new database 'studio_foto_cekrek'
- Import database_clean_original_[timestamp].sql (the CLEAN version)

### ❌ CSS Not Loading
Problem: Website looks unstyled
Solution:
- Check if assets/css/style.css exists
- Verify folder structure is intact
- Check browser console for 404 errors

### ❌ Database Connection Error
Problem: "Database connection failed"
Solution:
- Verify config/database.php settings
- Ensure MySQL service is running
- Check database name is exactly 'studio_foto_cekrek'

### ❌ Per-Person Calculation Not Working
Problem: Grup package doesn't show people count field
Solution:
- Verify database import was successful
- Check if packages table has 'includes' column
- Grup package should have "Harga per orang" in includes field

### ❌ Duration Shows in Hours
Problem: Packages show "1 jam" instead of "60 menit"
Solution:
- Check if packages table has 'duration_minutes' column
- Re-import the clean database
- Verify duration_minutes values are set correctly
